# Appendix G. Implementation Evidence Map

## How to Use This Map

This appendix connects the book's concepts to the companion implementation. Use
it when a chapter feels clear in prose but you want to know where the idea is
proved in code, SQL, tests, or operational queries.

Read each row as a chain:

```text
concept -> implementation artifact -> validation evidence -> operator proof
```

A serious production book should not leave reliability as a belief. It should
show where the claim becomes executable.
